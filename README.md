



# Estrutura SWITCH CASE em JavaScript

  Este repositório contém três exercícios desenvolvidos em JavaScript utilizando a estrutura de controle `switch case`.
  O objetivo é praticar a tomada de decisão com base em escolhas do usuário.





# 1. Seleção de Estilo de Filme Favorito

  O programa apresenta uma lista de estilos de filmes e solicita que o usuário escolha uma opção.

* 1 - Ação
* 2 - Comédia
* 3 - Drama
* 4 - Terror
* 5 - Ficção Científica
* 6 - Romance

# Seleção de Gênero Musical Preferido

O programa permite ao usuário escolher seu gênero musical favorito.

# Opções disponíveis:

* 1 - Rock
* 2 - Pop
* 3 - Hip Hop
* 4 - Eletrônica
* 5 - Jazz
* 6 - Clássica

# Funcionalidades:

* Entrada de dados do usuário
* Uso de `switch case`
* Exibição da mensagem:

  
Seu gênero musical preferido é: [gênero]
  


# Seleção de Tema Personalizado

Neste exercício, o usuário escolhe um tema de interesse para explorar.

# Exemplo de opções:

* 1 - Programação
* 2 - Design
* 3 - Jogos
* 4 - Inteligência Artificial
* 5 - Desenvolvimento Web

# Funcionalidades:

* Permite escolha de tema
* Exibe mensagem personalizada para cada tema
* Incentiva o desenvolvimento na área escolhida
* Trata entradas inválidas



# Tecnologias Utilizadas

* JavaScript
* HTML (para interação com o usuário, se aplicável)
